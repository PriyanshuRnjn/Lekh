## ✨ About LEKH

Managing a small business should be simple.
LEKH is designed to remove the complexity of traditional accounting software and provide a clean, fast and offline-first experience.

Built with a strong focus on usability and performance, LEKH helps shop owners handle daily operations from one place.

---

## 🧾 Core Features

### Billing & Invoicing

* Create invoices in seconds
* Keyboard-friendly billing workflow
* Generate professional printable PDF bills
* One-click printing support
* Fast item lookup using item codes

### 📦 Inventory Management

* Real-time stock tracking
* Automatic stock reduction after billing
* Low stock alerts and reorder suggestions
* Daily stock insights and analytics

### 👥 Customer Management

* Store customer purchase history
* Track last visit and buying patterns
* Identify top customers by value and frequency
* Maintain long-term customer records

### 📊 Dashboard & Reports

* Daily and monthly sales overview
* Quick business performance insights
* Key metrics at a glance

### 💾 Offline First

* Works completely offline
* No internet required
* All data stored locally on the device

## 🛠️ Tech Stack

LEKH is built using modern and reliable technologies:

* **Python** — Core application logic
* **CustomTkinter** — Desktop UI framework
* **SQLite** — Local database storage
* **HTML + CSS** — Invoice templates
* **Node.js** — High-quality PDF generation
